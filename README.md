# 🔢 Simple-Math-Game-Kotlin
A fun and interactive math practice game built with Kotlin.

This is a dynamic Android application developed as part of my **Computer Programming** studies. The project focuses on implementing a custom mathematical operation algorithm with a modern user interface.

## 🚀 Key Technical Features
* **PEMDAS Algorithm:** Features a custom-built logic to handle mathematical operation priority (multiplication/division before addition/subtraction) using a list-filtering approach.
* **Modern UI:** Entirely built using **Jetpack Compose**, utilizing declarative UI components like Cards, OutlinedTextFields, and Material3 themes.
* **Input Validation:** Implements real-time input filtering to ensure only numeric values, decimal points, and minus signs are processed.
* **Zero-Division Protection:** Includes safety checks to prevent application crashes during division operations.

## 🛠 Tech Stack
* **Language:** Kotlin
* **Framework:** Jetpack Compose (Material Design 3)
* **Architecture:** Clean Code principles with state management using `remember` and `mutableStateOf`.
